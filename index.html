Veritas is an LLM fine-tuned to flag linguistic markers of deception in written statements — hedging, distancing language, inconsistent detail — and return a confidence score, not a verdict.

SIGNAL: LINGUISTIC_TRACE.LOG STATUS: MONITORING
01 — Overview

Read the statement, not the person

Veritas doesn't watch faces or measure pulse. It works entirely from text — transcripts, chat logs, written statements — scoring each claim against patterns associated with deceptive language in prior research: qualifier overuse, low self-reference, verb tense shifts, missing sensory detail.

02 — How it works

Three passes over every statement

IN

Statement intake

Raw text is chunked into individual claims so each one can be scored independently rather than the passage as a whole.

02

Linguistic scan

The model checks each claim for known deception markers — hedges, distancing pronouns, tense inconsistency, over-specific or under-specific detail.

03

Confidence score

Flagged claims are weighted into a single 0–100 score with the contributing phrases highlighted, so the reasoning stays visible.
